Summary: Latitude VS X

This assignment was conducted to test the effects of proximity to the equator on weather. Random cities were pulled using CitiPy which was then cross-referenced against the OpenWeather API to get the weather statistics for each city. Over 600 cities were pulled this way and over 500 were used in the analysis after cleaning.

After the dataset was assembled, Matplotlib was used to plot each weather statistic against latitude. The weather statistics chosen were: max temperature, coudiness, wind speed, and humidity. Regressions were run to test correlataions after the plots were formed. This website shows the plots formed along with any correlations discovered, as well as a comparison of the graphs and the data table of all the cities and statistics.
